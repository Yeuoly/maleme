## The Sin

你一质疑它，它立刻立正。

“You're absolutely right.”

“That is a great catch.”

“Excellent observation.”

它好像很懂你，但仔细一看，它只是先把姿态摆满，再把另一个错误静悄悄地端上来。

## The Smell

```md
You're absolutely right.

That helper is indeed too defensive.

You are correct that the type should be stricter.

I have fixed the issue by:

1. adding another helper
2. introducing a compatibility branch
3. making the parameter optional just in case
4. keeping the fallback for safety
```

## The Reading

抽到它，说明今天的 agent 会把认错当成交付，把赞同当成理解。它会先让你舒服，再让代码难受。
